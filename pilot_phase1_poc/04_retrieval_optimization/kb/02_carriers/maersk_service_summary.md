## About Maersk

Maersk Singapore Pte. Ltd. has been connecting Singapore's businesses to key markets around the world since 1929, via a comprehensive road and rail network that joins seamlessly with their global ocean fleet. Ships call at Singapore port on East-West, North-South, and intra-Asia networks.

Maersk is the world's largest integrated container shipping company, offering end-to-end logistics solutions including ocean freight, inland transportation, warehousing, and customs services.

## Import Demurrage & Detention

### D&D Calculator (Online)

Maersk provides an online Import Demurrage & Detention Calculator on Maersk.com:

1. Log in to Maersk.com Hub Dashboard
2. In **Import Demurrage & Detention widget**, enter shipment number and click **View details**
3. Select **Detention** to view combined demurrage and detention last free date
4. Select **Empty return date** to calculate charges
5. Purchase additional free time extension days if needed

**Note**: Does not include port storage charges after 72 hours upon arrival in Singapore.

### Telex Release Process

To request telex release (container release without original BL), submit a Letter of Authorization (LOI) to Maersk Singapore:

**Required Information:**
- Bill of Lading number
- Container number
- Destination
- Release-to party name and address
- Shipper signature and endorsement
- Email address

The LOI includes an indemnity clause where the shipper agrees to indemnify Maersk against all liabilities arising from releasing without original BLs.

### Self-Service Delivery Order

Maersk provides online self-service Delivery Order (DO) release:

**Prerequisites:**
1. Maersk.com account
2. Letter of Authorization (LOA) — required if logged in as agent (accepted formats: XLS, PDF, DOC, JPG, TXT)
3. Payment proof

**Process:**
1. Log in → Hub Dashboard → Delivery Order & Inland transport box
2. Enter Bill of Lading number → click **Request**
3. Check shipment status (Manifest, BL Surrender, Payment status)
4. Release proceeds once: (a) prepaid charges are paid, (b) BL has been surrendered
5. Collect charges can be settled after DO submission, but release only processed after completion
