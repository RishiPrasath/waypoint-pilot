Guide to Customs' Procedures
Module 5: TradeNet® Permit Applications

**Important Permit Fields**

**Inward Applications**

|S/N|Key fields required for<br>Inward Applications|Description|
|---|---|---|
|1.|BG INDICATOR|Reference code used to indicate which Banker's Guarantee is to be<br>used.<br>If "D" is selected or if the field is left blank, declaring agent's<br>Banker's Guarantee will be used.<br>If "I" is selected, importer's or exporter's Banker's Guarantee will be<br>used.|
|2.|BRAND NAME|Tradename or the brand of the goods.|
|3.|CA LICENCE NUMBER|Licence Number issued by the relevant Controlling Agency, if any.|
|4.|CA/SC CODE 1|Information required by CAs and/or Customs for certain goods or<br>permit applications e.g.<br>For strategic goods, indicate the end-use code 1.<br>For motor vehicles, indicate the engine number.<br>For seastores, indicate the number of crew.|
|5.|CA/SC CODE 2|Information required by CAs and/or Customs for certain goods or<br>permit applications e.g.<br>For strategic goods, indicate the end-use code 2.<br>For motor vehicles, indicate the chassis number.<br>For seastores, indicate the voyage duration.|
|6.|CA/SC CODE 3|Information required by CAs and/or Customs for certain goods or<br>permit applications e.g.<br>For strategic goods, indicate the end-use code 3.<br>For motor vehicles, indicate the vehicle type - "N" for new vehicles,<br>"U" for used vehicles or "T" for Taxi.|
|7.|CA/SC PRODUCT CODE|Refers to product code assigned by Controlling Agencies (CAs)<br>and/or Customs for certain goods or permit applications e.g.<br>- motor vehicle product codes|
|8.|CA/SC PRODUCT CODE<br>QUANTITY AND UNIT|Indicate product code quantity and unit for goods, if required by CAs<br>or Customs|
|9.|CARGO PACKING TYPE|Cargo packing type for the cargo (e.g. containerised).|
|10.|CLAIMANT CODE|Applicable for duty exemption and/or GST Relief applications.<br>Indicate the code assigned to an approved or registered claimant.|
|11.|CLAIMANT NAME|Applicable for duty exemption and/or GST Relief applications.<br>Indicate the name of the person from an entity who is authorised to<br>claim exemption from Customs duty and/or GST relief on the goods,<br>on behalf of the entity.|
