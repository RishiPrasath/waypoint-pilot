## 1. Overview

Singapore has one of the most extensive FTA networks globally, with 27 implemented agreements covering 40+ trading partners. This matrix helps identify the optimal FTA for preferential tariff treatment.

---

## 2. Major FTAs Summary

### 2.1 Regional Agreements

| Agreement | Partners | Effective | Key Feature |
|-----------|----------|-----------|-------------|
| **AFTA/ATIGA** | ASEAN 10 | 1993/2010 | 98.86% tariff-free |
| **RCEP** | ASEAN+5 (CN, JP, KR, AU, NZ) | 2022 | Cumulation across 15 |
| **CPTPP** | 11 countries | 2022 (SG) | High standards, broad coverage |

### 2.2 Bilateral Agreements

| Agreement | Partner | Effective | Key Feature |

| **USSFTA** | USA | 2004 | Zero tariff on 99% |

## 7. RCEP vs ATIGA vs CPTPP

### 7.1 Feature Comparison

| Feature | ATIGA | RCEP | CPTPP |
|---------|-------|------|-------|
| **Members** | 10 (ASEAN) | 15 | 11 |
| **Singapore coverage** | All ASEAN | ASEAN+5 | Overlaps RCEP |
| **Tariff elimination** | 98.86% | 91-92% | 99%+ |
| **Timeline** | Completed | 20 years | 11 years |
| **Cumulation** | ASEAN only | All 15 | All 11 |
| **Self-certification** | No | Optional | Yes |
| **E-commerce rules** | Basic | Enhanced | Comprehensive |
| **Labor/Environment** | No | Basic | Comprehensive |

### 7.2 When to Use Each

| Scenario | Recommended FTA |
|----------|-----------------|
| ASEAN-only supply chain | ATIGA |
| Inputs from China/Japan/Korea | RCEP |
| High-standard market access | CPTPP |
| Need cumulation across regions | RCEP |
| E-commerce focus | RCEP or CPTPP |
| Services trade | CPTPP or bilateral |

## 8. Documentation Checklist by FTA

### 8.1 ATIGA (Form D)

- [ ] Commercial invoice
- [ ] Packing list
- [ ] Bill of lading/Air waybill
- [ ] Form D application
- [ ] Cost breakdown/RVC calculation
- [ ] Manufacturing records (if required)
- [ ] HS classification evidence

### 8.2 RCEP

- [ ] Commercial invoice
- [ ] Packing list
- [ ] Transport documents
- [ ] RCEP Certificate of Origin or Declaration
- [ ] Origin criteria demonstration
- [ ] Cumulation records (if applicable)

### 8.3 CPTPP

- [ ] Commercial invoice
- [ ] Certification of Origin (self-declaration)
- [ ] HS classification
- [ ] Origin documentation
- [ ] Importer/producer certification

### 8.4 EUSFTA

- [ ] Commercial invoice
- [ ] EUR.1 or Origin declaration (REX)
- [ ] Statement on origin
- [ ] Supplier declarations (if cumulation)

## 9. Common Pitfalls

| Pitfall | Impact | Prevention |
|---------|--------|------------|
| Wrong FTA selection | Higher duties paid | Use comparison matrix |
| Incorrect HS code | CO rejection | Verify classification |
| Insufficient RVC | Origin denied | Calculate before shipping |
| Missing documentation | Delays, duties | Use checklist |
| Expired CO validity | Not accepted | Check validity periods |
| Direct consignment broken | Lose preference | Use through B/L |

## 11. FTA Utilization Decision Flowchart

```
START
  │
  ├─ Is destination in ASEAN?
  │     └─ YES → Use ATIGA (Form D)
  │
  ├─ Is destination China/Japan/Korea?
  │     └─ YES → Compare RCEP vs Bilateral
  │           └─ Choose lower duty rate
  │
  ├─ Is destination USA/EU/UK?
  │     └─ YES → Use specific bilateral
  │
  ├─ Is destination CPTPP member?
  │     └─ YES → Compare CPTPP vs others
  │
  └─ Multiple FTAs available?
        └─ YES → Compare:
              1. Tariff rate
              2. RoO complexity
              3. Cumulation needs
              4. Documentation burden
```
