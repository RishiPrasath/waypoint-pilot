## Digital Solutions

PIL provides comprehensive digital tools for customers:

| Tool | Function |
|------|----------|
| **Track & Trace** | Real-time cargo tracking |
| **Schedule Finder** | Search by Port, Point-to-Point, or Vessel |
| **PocketPIL** | Customer portal for bookings and documentation |
| **eVGM** | Electronic Verified Gross Mass submission |
| **e/BL** | Electronic Bill of Lading |
| **Data Integration** | EDI/API connectivity |
| **Carbon Calculator** | Online emissions calculator |

### Schedule Search Options

1. **By Port** - View all sailings from a specific port
2. **Point 2 Point** - Search routes between origin and destination
3. **By Vessel** - Track specific vessel schedules

## Documentation Requirements

### Digital Documentation

- **eVGM Submission**: Electronic VGM filing through PocketPIL or eVGM portal
- **e/BL**: Electronic Bill of Lading available
- **Shipping Instructions**: Submit via PocketPIL customer portal

### Standard Documentation

- Bill of Lading (paper or electronic)
- Commercial Invoice
- Packing List
- VGM Declaration (SOLAS requirement)

## Tariffs and Charges

PIL provides transparent tariff information through their website:

- **Tariffs & Charges Search**: Online lookup by port/route
- **Detention & Demurrage (D&D)**: Published rates by location
- **Additional Free Time**: Information available by request

## Key Strengths for Singapore Shippers

1. **Singapore-Based**: Headquarters in Singapore with deep local expertise
2. **SEA Specialist**: Largest home-grown carrier in Southeast Asia
3. **Extensive Regional Network**: Strong coverage across ASEAN markets
4. **Digital-First**: Comprehensive e-commerce tools (eVGM, e/BL, PocketPIL)
5. **Partner Network**: Access to specialized routes via MELL and MSCorp
6. **Multi-Cargo Capability**: Handles dry, reefer, breakbulk, and special cargo

## Booking Process

1. Register on **PocketPIL** customer portal
2. Search schedules via Point-to-Point or By Port
3. Submit booking request with cargo details
4. Receive booking confirmation
5. Submit Shipping Instructions before cutoff
6. File VGM via eVGM portal
7. Track shipment via Track & Trace
